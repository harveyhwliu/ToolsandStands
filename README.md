# 002ProtocolBuffer
&emsp;&emsp;Protocol Buffers，简称Protobuf，是谷歌开源的一项序列化技术，跨语言，跨平台，相比XML和JSON 更小、更快、更容易，因为XML、JSON为了可阅读、自解释被设计成字符文本形式，所以体积更大，在编码解码上也更麻烦，而Protobuf序列化为binary stream，体积更小，但是丧失了可读性`(现网实际是如何查看的呢？ od -c data.output ?)`。
